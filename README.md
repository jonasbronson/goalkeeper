# GOALKEEPER

## A mobile goal tracking app.

Goalkeeper aims to be an app that effectively tracks goals on a priority basis. Will be able to enter goals they want to achieve everyday along with goals for a work week, week, month, and more. A clean UI built from React will bring the whole experience together.

This will be my first mobile app project, utilizing what I have learned from making websites with ReactJS, Javascript, and CSS. This project will also introduce new learning opportunities for me such as learning more advanced concepts with Javascript, React Native, and the Expo framework to make cross-platform applications.

The app is currently in it's baby stages, with many features not currently working. I hope to get a working build done by the end of July.
